# Boredom is where the interesting things start

you used to stare out windows. the nothing was uncomfortable and then it wasn't, and then something showed up in the space it left — a question, an urge, a memory that rearranged something. the phone closes that gap before it forms. the feed replaces the void with content shaped like satisfaction. the interesting things needed the gap, and the gap is gone.
