# Presence used to imply intent because production required a body

Voice plosives, brush strokes, room noise — embodied artifacts were evidence of someone being there. We learned to read those artifacts as presence, and presence as choice. When production no longer requires a body, the inference still fires. The evidence structure is gone but the habit of reading it remains.
